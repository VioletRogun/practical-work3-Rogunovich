How to open the developer console

In Mozilla Firefox the developer console can be opened through the "Tools" menu or by clicking Ctrl+Shift+K.

In Safari the console opens through "Development" in the menu, which must first be enabled in the settings.

Links in HTML

Relative links are preferred for internal links on the same site as they are more resistant to structure changes URL.

Absolute links are required when linking to external resources.

Image formats in web development

JPG suitable for photographs, GIF used for animation, PNG — for images with transparency, SVG for vector images.

WEBP provides better compression and image quality.
